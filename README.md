# 异世界重开模拟器 · App 版

这是把游戏打包成了一个 **PWA（Progressive Web App / 渐进式网页应用）**。PWA 是目前不需要应用商店审核、不需要专门的开发者账号，就能让网页"变成一个真正的 App"的标准方式——装上以后有独立图标、独立窗口（没有浏览器地址栏），支持离线打开，安卓上和普通 App 几乎没有区别，iOS/桌面端也都支持。

## 文件说明

```
index.html                  游戏本体
manifest.json                App 的名称、图标、主题色等信息
sw.js                        Service Worker，负责离线缓存
icons/icon-192.png           小图标
icons/icon-512.png           大图标
icons/icon-512-maskable.png  安卓自适应图标（会被系统裁切成圆形/圆角矩形等形状）
```

## ⚠️ 重要：必须先"上线"，才能真正"安装"

PWA 的安装能力（浏览器地址栏出现"安装"按钮、离线缓存生效）**只有在通过 `http://` 或 `https://` 访问时才会生效**。如果你只是双击 `index.html` 用 `file://` 方式打开，游戏能正常玩，但浏览器不会提供"安装"选项，这是所有浏览器的统一限制，不是这份代码有问题。

所以要把它变成手机/电脑上真正可以"安装"的 App，你需要先把这个文件夹传到网上，哪怕是最简单的免费静态托管都够用。以下几种任选一种，几分钟就能搞定：

### 方式一：GitHub Pages（推荐，完全免费）
1. 在 GitHub 上新建一个仓库，把这个文件夹里的所有文件上传进去（保持 `icons/` 子文件夹结构不变）
2. 进入仓库的 Settings → Pages，Source 选择 `main` 分支、根目录，保存
3. 等一两分钟，GitHub 会给你一个形如 `https://你的用户名.github.io/仓库名/` 的网址
4. 用手机或电脑浏览器打开这个网址

### 方式二：Netlify / Vercel（也免费，且不需要会用 Git）
把整个文件夹直接拖拽到 [app.netlify.com/drop](https://app.netlify.com/drop) 这样的页面上，几秒钟就能拿到一个可访问的网址。

### 方式三：自己电脑上临时起一个本地服务器（仅用于自己测试）
如果电脑装了 Python，在这个文件夹里打开终端执行：
```
python3 -m http.server 8000
```
然后浏览器访问 `http://localhost:8000` 即可体验完整的安装流程（仅限当前这台电脑访问，不能分享给别人）。

## 安装到主屏幕 / 桌面

网址部署好之后：

- **安卓 (Chrome)**：打开网址，地址栏或菜单里会出现"安装应用"/"添加到主屏幕"，点一下即可，桌面上会出现独立图标
- **iPhone/iPad (Safari)**：打开网址 → 点击底部分享按钮 → "添加到主屏幕"
- **电脑 (Chrome / Edge)**：打开网址，地址栏右侧会出现一个"安装"图标，点一下就会变成一个独立的桌面应用窗口

安装之后再打开就完全脱离浏览器界面了，图标、窗口标题都是"异世界重开模拟器"，而且因为有离线缓存，就算没网也能正常开新的一局。

## 如果你只是想要个"快捷方式"，不想折腾部署

不追求离线缓存和正式的安装流程的话，其实不用做任何这些配置：直接用 Chrome 打开原来那个单文件 `index.html`，点右上角菜单 → "更多工具" → "创建快捷方式"，勾选"以窗口方式打开"，就能在桌面上生成一个图标，双击后以独立窗口打开，观感上已经很接近一个 App 了，唯一区别是没有离线缓存。
